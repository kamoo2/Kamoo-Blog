## Problem

> 아이언슈트를 입고 N위치로 이동하는데 사용되는 건전지 사용량을 구하자!

1. 아이언슈트는 K거리 만큼 점프할 수 있다! (건전지 사용량 K)
2. 아이언슈트는 현재 위치의 두배인 위치로 순간이동이 가능하다! (건전지 사용량 0)

<hr />

## Solve

처음 이 문제를 보고 완전탐색으로 풀어야 하나 ? 라고 생각했지만 N의 범위가 너무 컸기에 불가능 했다.

번뜩 현재 위치의 x2 위치로 순간이동 하는 방법은 사용량이 0이니깐 결국 8로 이동할 때나 4로 이동할 때나 2로 이동할 때나 사용량이 같다는 점이 떠올라, 손으로 짝수라면 계속 나누고 홀수라면 1을 빼주고 다시 나눠지도록 풀어보았다.

이때 나오는 홀수의 개수가 전기사용량이 되는 것을 확인할 수 있었고, 이는 10진수를 2진수로 바꿨을 때 1의 개수와 같아, 10진수를 2진수로 바꾼 후에 문자열을 배열로 변환한 다음 돌면서 1의 개수를 세는 방식으로 풀었다.

### 10진수를 2진수로 바꾸는 방법

`number.toString(2)`

<hr />

## Code

```javascript
function solution(n) {
  let ans = 0;

  // 10진수를 2진수로 변경
  n.toString(2)
    .split('')
    .forEach((item) => {
      if (item === '1') ans++;
    });

  return ans;
}
```
